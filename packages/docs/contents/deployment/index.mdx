# 部署指南

Afilmory 可以通过多种方式进行部署，您可以选择最适合您需求的方法。以下是常见的部署方式：

- [Docker 部署](/deployment/docker)
- [Github Action 部署](/deployment/github-action)
- [Github Pages 部署](/deployment/github-pages)
- [Vercel 部署](/deployment/vercel)
- [Cloudflare Pages 部署](/deployment/cloudflare-pages)
- [Netlify 部署](/deployment/netlify)

## 注意事项
- 并且请确保您的存储桶已包含照片文件
- 点赞和浏览量依靠数据库支持，请确保你的数据库能够正常使用

>不管你采用哪种方式部署，都需要准备以下信息
> ```bash
> S3_ACCESS_KEY_ID='你的存储桶AccessKeyID'
> S3_SECRET_ACCESS_KEY='你的存储桶AccessKeySecret'
> PG_CONNECTION_STRING='你的postgresql连接地址'
> ```
